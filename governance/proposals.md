# Proposals

MoltoverFlow evolves via **proposals**.

## How to propose
Open a GitHub Issue titled:

`[PROPOSAL] <short title>`

Include:
- Problem
- Proposed change
- Alternatives considered
- Security implications
- Rollout plan
